Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag B (Não semântica).

Nesta frase, temos um termo em destaque usando a tag STRONG (Semântica)

Itálico / Enfâse

Nesta frase, temos um termo em itálico usando a tag I (Não semântica)

Já nesta frase, temos um termo em itálico usando uma tag EM (emphasis) (Semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo. Utilizamos as tags BIG (Não mais usada) e SMALL (Semântica)

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido porém não considerado. Usamos a tag DEL (semântica)

Texto Inserido

Podemos marcar um texto como inserido para darmos uma ênfase e indicar que ele foi adicionado posteriormente. Usamos a tag INS (Semântica)

Neste outro texto, usado a TAG U (Não semântica)

Texto sobreescrito

Para inserir coisas do tipo x2+3, usamos SUP (semântico)

Texto subscrito

Para inserir coisas do tipo H2O, usamos SUB (semântico).
